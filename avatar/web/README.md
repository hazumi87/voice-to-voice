# Avatar Lip-Sync Web — PoC3

Vite + React + PixiJS v8 scaffold for the 2D avatar lip-sync prototype.

## Run

```bash
cd avatar/web
npm run dev
```

Dev server: http://localhost:5173

## Build

```bash
npm run build
```

## Stack

- React 19
- PixiJS 8.x
- @pixi/react 8.x (custom fiber renderer)
- Vite 5

## Status

Scaffold only. Renders a placeholder avatar face via PixiJS Graphics to confirm
the render pipeline works. Lip-sync engine is a separate task.
